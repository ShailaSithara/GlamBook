# 🌸 About GlamBook

**GlamBook** is an all-in-one beauty service and rental management platform developed using **Flutter** and **Python**. It connects customers with makeup artists and rental shops while providing an efficient admin panel for complete system management.

The application simplifies the entire booking process by allowing users to discover beauty professionals, rent fashion accessories, book appointments, manage orders, and track bookings—all within one elegant platform.
